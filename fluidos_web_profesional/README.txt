
Proyecto web académico: Viscosidad, Reynolds, Poiseuille y Stokes
=================================================================

Archivos:
- index.html : estructura del sitio
- styles.css : estilos visuales
- script.js  : calculadoras y laboratorios interactivos

Uso:
1) Descomprima el archivo ZIP.
2) Abra index.html en cualquier navegador moderno.
3) No requiere instalación.

Recomendación para entrega:
- Si el docente pide repositorio, subir estos archivos a GitHub.
- Si pide publicación web, se puede desplegar en GitHub Pages o Netlify.
- Revise y, si desea, agregue portada con nombre del estudiante, curso, universidad y docente.
